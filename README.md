# 📝 Exercism Solutions

This repository contains my solutions to [Exercism](https://exercism.org/) coding exercises.  
All submissions are automatically synced from my Exercism account via the [Exercism Solutions Syncer](https://github.com/apps/exercism-solutions-syncer) bot.

## 🗂️ Structure

Solutions are organized by **programming language**:

| Language   | Directory                     | Progress  |
| ---------- | ----------------------------- | --------- |
| JavaScript | [`/javascript`](./javascript) | ✅ Active |
| PHP        | [`/php`](./php)               | ✅ Active |
| TypeScript | [`/typescript`](./typescript) | ✅ Active |

Each directory contains subdirectories for individual exercises, named after the exercise slug (e.g., `resistor-color-trio`, `line-up`).

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Rockvald/Exercism.git
cd Exercism
```

### Run a Solution

Navigate to the exercise directory and follow the language-specific instructions.  
You can find the instructions in the `HELP.md` file of the exercise directory.
